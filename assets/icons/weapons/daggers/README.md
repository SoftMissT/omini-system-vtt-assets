# ⚔️ Weapons - Daggers

Ícones de daggers em todas as raridades.

## Arquivos (6 ícones)

- `icon_dagger_common.webp` - Common (Cinza #9E9E9E)
- `icon_dagger_rare.webp` - Rare (Azul #2196F3)
- `icon_dagger_epic.webp` - Epic (Roxo #9C27B0)
- `icon_dagger_legendary.webp` - Legendary (Dourado #FFD700)
- `icon_dagger_mythic.webp` - Mythic (Arco-íris Prismatic)
- `icon_dagger_cursed.webp` - Cursed (Preto/Vermelho #000000 + #FF2B4A)

## Sistema de Raridade

As cores seguem o padrão de raridade do sistema:
- Quanto mais raro, mais intenso o brilho
- Mythic tem efeito prismático (arco-íris)
- Cursed tem aura corrompida (sombra + vermelho)
